# Library Management System with Book Recommendations

## Описание проекта

Веб-приложение для управления библиотекой с интеллектуальной системой рекомендаций книг. Приложение реализует полноценный REST API с аутентификацией пользователей, CRUD операциями и алгоритмической системой рекомендаций на основе гибридного подхода.

## Функциональность

### 1. Управление пользователями
- Регистрация новых пользователей
- Аутентификация с JWT токенами
- Профиль пользователя

### 2. Управление книгами (CRUD)
- Добавление новых книг
- Просмотр списка книг
- Получение информации о книге по ID
- Обновление информации о книге
- Удаление книг

### 3. Категории книг
- Создание категорий
- Фильтрация книг по категориям

### 4. Отзывы и оценки
- Добавление отзывов к книгам
- Рейтинг книг (1-5 звезд)
- Автоматический расчет среднего рейтинга

### 5. Система выдачи книг
- Выдача книг пользователям
- Возврат книг
- Отслеживание срока возврата
- История выдачи

### 6. Система рекомендаций (Бизнес-логика)
Гибридный алгоритм рекомендаций, объединяющий:
- **Коллаборативную фильтрацию** - поиск похожих пользователей на основе их оценок
- **Контент-фильтрацию** - рекомендация книг, похожих на прочитанные пользователем
- **Взвешенную оценку** - учет временного фактора и популярности

Алгоритм учитывает:
- Историю чтения и выдачи книг
- Оценки и отзывы пользователя
- Предпочтения по авторам и жанрам
- Временной распад для старых взаимодействий

## Технологии

- **FastAPI** - веб-фреймворк
- **PostgreSQL** - реляционная база данных
- **SQLAlchemy** - ORM
- **JWT** - аутентификация
- **Pytest** - тестирование
- **Docker & Docker Compose** - контейнеризация

## Установка и запуск

### Через Docker (рекомендуется)

```bash
# Клонирование репозитория
git clone <repository-url>
cd library_app

# Запуск через Docker Compose
docker-compose up --build
```

### API Документация

После запуска приложения документация доступна по адресам:
 - Swagger UI: http://localhost:8000/docs

 - ReDoc: http://localhost:8000/redoc
